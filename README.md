## ðŸ›  Technical Overview
* *Frontend:* Built with HTML5 and CSS3. We used a mobile-first approach to meet the course requirements for responsiveness.
* *Database:* Instead of a traditional server, we used the *Web Storage API (LocalStorage)*. This allows the menu and orders to be saved directly in the browser.
* *Security:* We used sessionStorage to protect the admin.html page. If a user is not logged in as an admin, the system redirects them to the login page.
